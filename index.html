Custom Crafts

Custom Crafts is a full-stack e-commerce application where you can shop for creatively "custom" designed spaceships. The storefront app was built utilizing React.js, Python 3, GraphQL, Auth0, and 3D interactive models that leverage Google's Model Viewer technology. Users may sign-in, view, add-to-cart, write reviews, star, purchase and become return customers with profile and purchase history features. Customers may shop for different and varying styles of spacecrafts ranging from performance-build (i.e. racing), luxury, military, etc., all designed to suit their taste and whet their palates!

Facts 'n Flags

Facts 'n Flags is a small simple informative and fun front-end client-side application powered by React.js and using React Router, React Hooks, CSS Bootstrap 4 and CSS Flexbox. It a website where you decide which of the fifty United States of America you want to learn a little more about. The facts for each state are being fetched by an external API which include: Name of US State, its capital city, its largest city, its population, and its flag displayed in color! Enjoy!
